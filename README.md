# Tiny Shop Pixel Sim – Phaser + TypeScript

**Version:** v0.1.0 • 2025-10-28

## Stack
TypeScript + Vite + Phaser 3.60.x • Desktop & Mobile Browser • Pixel‑Art 16px

## Dev
```bash
npm i
npm run dev
npm run build
npm run preview
```

### GitHub Pages
- `npm run build` → `dist/`
- Falls Repo‑Pfad (z. B. `/tiny-shop`): `GH_PAGES_BASE=/tiny-shop npm run build`
